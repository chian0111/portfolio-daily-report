# 📈 Portfolio Tracking System: 投資理財的數位實驗室

這是為自己量身打造的投資組合追蹤工具。在研究美股與半導體產業的過程中，我發現「數據的即時性」與「視覺化整理」對於理性決策至關重要。

### 💡 開發初衷
投資不該只憑感覺，更應建立在邏輯及數據之上。我希望透過此專案，將投資數據（Portfolio Data）進行自動化彙整，使我能從瑣碎的數字記錄中抽離出來，專注於產業趨勢的研判。

### ✨ 專案核心功能
* **數據自動化彙整**：取代手動輸入，透過 `portfolio_system.py` 處理投資組合的數據邏輯。
* **快速環境部署**：利用 `setup.sh` 確保在不同設備上都能快速建立分析環境。
* **嚴謹的檔案管理**：透過 `.gitignore` 確保敏感的財務資料（如 `.json`, `.pdf`）留在本地，不被誤傳至雲端。

### 🛠️ 技術規格
* **Python**: 數據處理核心。
* **Git**: 數位資產的版本控制。
