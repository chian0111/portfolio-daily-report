# 📊 Portfolio Daily Report System

這是一個專為提升工作效率而開發的自動化報告系統，旨在將重複性的數據整理流程轉化為結構化的數位資產。

## 🎯 解決的工作痛點
在日常財會實務中，我發現「手動彙整多來源數據」與「重複性報告撰寫」是極大的隱形成本。
* **手動錯誤風險**：減少人工複製貼上導致的數據偏離。
* **時間碎片化**：將原本需耗時 30-60 分鐘的彙整工作，縮短至秒級自動化處理。
* **數據追蹤困難**：透過標準化的輸出格式，確保歷史報告具備高度的一致性與可追溯性。

## ✨ 主要功能
* **自動化整合**：一鍵彙整多來源的投資組合數據或工作日報。
* **結構化輸出**：自動生成符合審核需求的報告格式。
* **快速部署**：內置 `setup.sh` 腳本，支持快速環境搭建與執行。

## 🛠️ 技術工具
* **Language**: Python 3.x
* **Workflow**: Shell Scripting (`setup.sh`), Data Processing (`portfolio_system.py`)
